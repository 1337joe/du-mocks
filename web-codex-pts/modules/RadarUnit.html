getConstructOwner (id)
Returns a table with id of the owner entities (player or organization) of the given construct, if in range and if active transponder tags match.

Parameters:

  • id int The ID of the construct.

Returns:

    table A table { playerId: pID, organizationId: oID } describing the owner. Use system.getPlayerName(pID) and system.getOrganizationName(oID) to retrieve info about it.

See also:

  • system:getPlayerName
  • system:getOrganizationName

getConstructInfos (id)
Returns a list of working elements on the given construction, if identified.

  • id int The ID of the construct.

Returns:

    table A table { weapons: f, radars: f, antiGravity: f, atmoEngines: f, spaceEngines: f, rocketEngines: f } with values 0.0-1.0. Exceptionally antiGravity and rocketEngines are always 1.0 if present, even if broken.
